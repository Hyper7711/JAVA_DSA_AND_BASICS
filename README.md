# JAVA_DSA_AND_BASICS

A collection of Java programs covering **Core Java concepts**, **Data Structures**, **Algorithms**, and basic programming exercises. This repository is created for learning, practicing, and improving problem-solving skills using Java.

## 📚 Topics Covered

### Java Basics
- Variables & Data Types
- Operators
- Input/Output
- Conditional Statements
- Loops
- Methods
- Classes & Objects

### Data Structures & Algorithms
- Linear Search
- Binary Search
- Bubble Sort
- Array Operations
- Pattern Printing
- Prime Number Programs
- Basic Searching Algorithms

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/JAVA_DSA_AND_BASICS.git
```

### Navigate to the Project

```bash
cd JAVA_DSA_AND_BASICS
```

### Compile a Java File

```bash
javac FileName.java
```

Example:

```bash
javac BubbleSort.java
```

### Run the Program

```bash
java BubbleSort
```
