# JAVA_DSA_AND_BASICS

A beginner-friendly collection of **Java programs** covering **Core Java**, **Data Structures**, and **Algorithms**. This repository is built for learning, practicing, and improving programming and problem-solving skills.

## 📌 Topics
- Java Basics
- Arrays
- Searching (Linear & Binary)
- Sorting (Bubble Sort)
- Pattern Programs
- Prime Numbers
- Input/Output

## 🚀 Run a Program

```bash
javac FileName.java
java FileName
```

## 🎯 Purpose

Practice Java fundamentals and DSA concepts for coding interviews, academics, and competitive programming.
